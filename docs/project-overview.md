# Project Overview

## What This Project Is

Memory Match is a browser-based card matching game built with Next.js, React, TypeScript, and Tailwind CSS.

The player sees a grid of face-down cards. They flip two at a time trying to find matching pairs. The game ends when all pairs are found. Players are scored by how few moves they used and how fast they finished.

## Goals

1. Build a fully functional memory match game.
2. Learn professional React development practices through the process.
3. Demonstrate how a real frontend application is structured and built.

## Tech Stack

| Tool | Purpose |
|---|---|
| Next.js 15 | React framework with App Router |
| React 19 | UI component system |
| TypeScript | Type safety across all files |
| Tailwind CSS | Utility-first styling |
| Framer Motion | Card flip animations (Phase 10) |
| Local Storage | Persisting best scores |
| Vercel | Deployment |

## Core Features

- Card generation and shuffling
- Card flip interaction
- Match detection (two identical symbols = a match)
- Move counter
- Timer (starts on first flip)
- Best score storage per difficulty
- Three difficulty levels (easy / medium / hard)
