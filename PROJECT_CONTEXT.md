# Project Context

## Project Name

Memory Match Game

---

## Project Goal

Build a modern browser-based memory training game.

Players must flip cards and find matching pairs.

The project should be educational and demonstrate professional frontend development practices.

---

# Initial Tech Stack

Frontend:

* Next.js
* React
* TypeScript
* Tailwind CSS

Storage:

* Local Storage

Animation:

* Framer Motion (optional)

Deployment:

* Vercel

---

# Learning Objectives

The project owner wants to learn:

* React fundamentals
* State management
* Component architecture
* TypeScript
* Custom hooks
* Game logic implementation
* Animation systems
* Local storage persistence
* Frontend optimization
